"# TrLetter"

Используются:
Python 3.8
Django 3.2
PostgreSQL 10


=====================

Создание виртуального окружения Python:
python -m venv venv

Активация виртуального окружения:
venv\scripts\activate.bat

Установка пакетов:
pip install -r requirements.txt

Запуск:
cd trletter
python manage.py runserver

http://127.0.0.1:8000/
